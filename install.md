# 📱 안드로이드 앱(APK) 설치 가이드

개발 중인 앱을 안드로이드 폰에 직접 설치하는 방법입니다. **개발자 모드를 켤 필요 없이** 아래 설정만 확인하면 바로 설치가 가능합니다.

## 1. APK 파일 다운로드

- readme 페이지에 첨부된 최신 **app-release.apk** 파일을 스마트폰에서 클릭하여 다운로드합니다.

## 2. '출처를 알 수 없는 앱' 설치 권한 허용

안드로이드는 보안상 구글 플레이 스토어가 아닌 곳에서 받은 파일의 설치를 차단합니다. 처음 설치 시 아래와 같은 권한 허용이 필요합니다.

### ✅ 설정 방법

1. 다운로드한 파일을 실행하면 **"보안상의 이유로 이 소스의 알 수 없는 앱을 휴대전화에 설치할 수 없습니다"**라는 팝업이 뜹니다.
2. 팝업에서 **[설정]**을 누릅니다.
3. **[이 출처 허용]** 스위치를 **'켬'**으로 바꿉니다.
    - *만약 팝업이 안 뜬다면:* `설정` > `애플리케이션` > `특별한 접근` > `출처를 알 수 없는 앱 설치` > 해당 앱(크롬 또는 내 파일) 선택 후 허용

## 3. 앱 설치 및 실행

1. 권한 허용 후 뒤로 가기를 누르면 **"이 앱을 설치하시겠습니까?"**라는 문구가 뜹니다.
2. *[설치]**를 누르고 완료될 때까지 기다립니다.
3. 설치가 완료되면 **[열기]**를 눌러 앱을 실행합니다.
4. 가입후 기본레벨을 일반사용자입니다, 관리자(대윤)에게 레벨 업그레이드를 요청하세요.

---

## ⚠️ 설치 시 주의사항 (필독)

### 1. "무시하고 설치" 클릭 (Play 프로텍트)

설치 중 **"Play 프로텍트에 의해 차단됨"**이라는 경고가 뜰 수 있습니다. 이는 정식 출시 전인 개발용 앱이라 뜨는 일반적인 메시지입니다.

- **[세부정보 보기]** 화살표를 누른 뒤, 반드시 **[무시하고 설치]**를 눌러야 진행됩니다.

### 2. 기존 앱 삭제 후 재설치

만약 이전 버전의 앱이 이미 깔려 있다면, **기존 앱을 완전히 삭제한 후** 새 APK를 설치해야 오류가 나지 않습니다.
