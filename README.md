## 버전 0.2.0
- 기본 개발을 대부분 완료었습니다. 버전을 0.2 로 올립니다.
- 모든 컨테츠를 직접 수정할수 있습니다.
- 컨텐츠 수정후 처럼 로드시 잔상이 현상을 제거하여고 로드후 백그라운드로, 설정 이미지등을 다운 받도록되어 있습니다.
- 모든 음성도 앱 내에 포함되것이 아니고 인터넷 접속시 다운받도록 설정되었습니다.
- 음성 메세지도 변경할수 있도록하였습니다.(미비한 부분은 보강예정)
- 현 컨탠츠 변경은 출시후에도 자유롭고 필요에 따라 수정할수 있는 형태입니다. 계절에 따라 분위기 이미지 전환이 자유롭습니다.
